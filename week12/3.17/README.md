### CS공부에 대해

CS공부를 어떻게든 해야겠다. 알고리즘 스터디에서 계속 출제되고 있는 백트래킹, DFS, BFS, 동적계획법, 그리디, 정수론 등등\

그리고 오늘 특강을 통해 현업에서 일하시는 선배님께서 어떻게 CS공부를 했는지 잠깐 보여주셨는데, 노션에 차곡차곡 빼곡하게 쌓인 공부의 흔적들을 보고 피하지 말아야할 숙제임을 제대로 알게 되었다.
