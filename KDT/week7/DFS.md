# 그래프 탐색 알고리즘

시작 정점에서 간선을 타고 이동할 수 있는 모든 정점을 찾는 알고리즘

- 깊이 우선 탐색 (Depth-First Search, DFS)
- 너비 우선 탐색 (Breath-First Search, BFS)

## 깊이 우선 탐색(DFS)

스택의 개념을 활용한다.

- 모든 정점을 방문할 때 유리함
- 경우의 수, 순열과 조합 문제에서 많이 사용
- 너비우선보다 간단

## 너비 우선 탐색 (BFS)

큐의 개념을 활용한다.

- 모든 정점을 방문할 필요가 없을 때 유리함
- 최단 거리를 구하는 경우 유리함
